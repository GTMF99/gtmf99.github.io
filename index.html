DuinoMote

DuinoMote es una aplicacion desarrollada en Java y XML para el sistema operativo móvil Android, donde se hace uso de herramientas como Room y el proyecto de código abierto: usb-serial-android para comunicarse con una placa de desarrollo Arduino precargada con un sketch capaz de captar y emitir señales de control a través de luz infrarroja.

La misma permite al usuario crear controles virtuales, y, a su vez, botones virtuales para estos, con el fin de ofrecer flexibilidad al operador para personalizar su control remoto virtual.

Sistema de Inventario

El Sistema de Inventario realizado para la empresa Predictive Technologies, es un programa de escritorio enfocado a entornos Microsoft Windows y desarrollado en C# y utiliza Microsoft Access como motor de bases de datos.

El mismo fue utilizado para controlar el inventario de dicha empresa. Para ello, contaba con distintos modulos: Resumen de Inventario, Ajustes de Inventario, Articulos y Reportes.

Clínica del Dolor y Cuidados Paliativos

Este sitio Web fue desarrollador para la Clínica del Dolor y Cuidados Paliativos ubicada en Ecuador, San Lorenzo, con el fin de proveer a los pacientes e interesados la informacion necesaria referente a los servicios ofrecidos por la clínica. La misma fue desarrollada con Wordpress para garantizar una sencilla administracion del sitio y del contenido del mismo.
